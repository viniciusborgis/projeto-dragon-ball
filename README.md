# 🐉 Dragon Ball: Cloud Rider

### ☁️ Sobre o Projeto

Inspirado no clássico **Dragon Ball: Advanced Adventure** e na mecânica viciante do "Dino" do Chrome, este mini-game traz o Goku em sua Nuvem Voadora para um desafio épico de reflexos. O objetivo é simples: desvie das nuvens negras e busque o recorde mundial\!

Criado do zero com **HTML5, CSS3 e JavaScript**, o foco do desenvolvimento foi garantir fluidez, diversão e uma experiência totalmente responsiva. Desperte seu instinto e jogue agora\!

-----

### 🎮 Como Jogar

1.  Acesse o link do projeto (ou abra o `index.html`).
2.  Clique no botão **"Jogar"**.
3.  Use os comandos do teclado ou toque na tela para controlar o Goku:
      * **W / Seta para Cima**: Voar para cima.
      * **S / Seta para Baixo**: Voar para baixo.
4.  Evite as nuvens negras a todo custo\! A cada nuvem ultrapassada, sua pontuação aumenta.

-----

### 🚀 Tecnologias e Conceitos Aplicados

Este projeto foi uma excelente oportunidade para aplicar conceitos fundamentais de desenvolvimento web:

  * **HTML5 Semântico:** Utilização de tags como `<main>`, `<section>`, `<article>` e `<footer>` para melhor acessibilidade e SEO.
  * **CSS3 Avançado:**
      * `clamp()`: Para fontes e contornos responsivos que se ajustam a qualquer tela.
      * `@keyframes`: Animações fluidas para o fundo e obstáculos.
      * **Flexbox/Grid**: Layout centralizado e organizado.
  * **JavaScript (Vanilla):**
      * **Game Loop**: Gerenciamento de quadros com `requestAnimationFrame`.
      * **Lógica de Colisão**: Cálculo de *hitboxes* entre o Goku e as nuvens.
      * **Dificuldade Dinâmica**: O jogo acelera conforme você pontua.
